# 涂林旧作品档案馆

本仓库存储了约 300 张「东衙」时期的涂林作品扫描件，旨在保存和展示涂林的早期发展。

## 仓库结构

```
/data             # 作品目录
	|- banzhang   # 包含部分电子化原稿
        |- turs
        |- xzx-flag
	|- xiaoli
        |- 1.jpg
        |- ...
        |- 30.jpg
	|- xuzei
        |- 1.jpg
        |- ...
        |- 37.jpg
	|- yaojie
        |- 1.jpg
        |- ...
        |- 172.jpg

/site             # 列表网页
	|- ...

README.md
build.sh          # 部署脚本
```

## 浏览

在克隆本仓库到本地后，您可以：

- 直接通过目录阅读；
- 将 `/data` 目录移入 `/site`，然后在 `/site` 目录使用 `python -m http.server 8080` 启动网页服务器。

另外，您也可以使用 `wrangler.toml` 作为 Cloudflare Workers 配置，部署静态页面托管。
