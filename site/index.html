涂林旧作品档案馆

GitHub | 备用

欢迎访问「涂林旧作品档案馆」。本仓库存储了约 300 张「东衙」时期的涂林作品扫描件，旨在保存和展示涂林的早期发展。

维护

档案馆馆员「讹字」目前匿名负责维护本仓库和托管在 Cloudflare 上的站点。
仓库所有者（Tokenicrat 词元）受授权，代为发布。任何联系请使用主页邮件。

著作权

版权所有 © 涂林创作者 2021-2024

仓库中所有作品皆为涂林创作者的个人作品，未经许可，不得发布、再创作、售卖。特别地，以下两种行为是可以接受的：

  • 完整地 分叉仓库或制作镜像站点，并 注明原作者
  • 基于作品中角色（包括小丽、班长、徐贼、姚姐、涛妈猪、大黄狗等）二次创作，并 注明原作者
